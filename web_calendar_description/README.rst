========================
Web Calendar Description
========================

This module allows to add description into a calendar view depending of an ir.parameter
with the key referenced into the context of the action of the implied calendar view.

For exemple:
context of the action:
{'calendar_description': 'leave_request'}

and the ir parameter:

key - leave_request
value - some text for the users into this calendar view
